# 관제 시스템 고도화 기능 요구사항 제안서

## 1. 사운드 알림 기능 추가
* **배경 및 필요성:** 시각적 표현 외에도 낙상 및 호흡 이상 등의 위급 상황 발생 시 상황을 즉각적으로 인지할 수 있도록 청각적 경고가 필요함.
* **구현 방안:** HTML5 웹 오디오 API(`AudioContext`)를 활용하여 상태별(낙상/호흡이상) 경고음을 생성함. 브라우저의 자동 재생 차단 정책을 우회하기 위해 최초 대시보드 진입 시 유저가 '음성 알림 활성화' 버튼을 누르도록 유도하는 모달이나 상단 토글 UI를 배치함.

## 2. 브라우저 푸시 알림 기능 추가
* **배경 및 필요성:** 관제 대시보드 탭이 백그라운드에 있거나 운영자가 다른 모니터로 문서 작업 등을 수행하고 있을 때도 시스템 OS 차원에서 비상 상황을 놓치지 않고 전송해야 함.
* **구현 방안:** 브라우저 내장 `Notification API`를 연동하여 페이지 로드 시 사용자에게 알림 권한을 요청하고, 서버로부터 이상 이벤트 수신 시 웹 애플리케이션의 활성화 여부와 관계없이 OS 알림 배너를 띄우도록 처리함.

## 3. 화면 번인(Burn-in) 방지 및 UI 리프레시 로직 도입
* **배경 및 필요성:** 24시간 내내 고정된 관제 UI 화면을 켜놓을 경우 LCD나 OLED 모니터 소자에 영구적인 잔상이 남는 번인 현상이 발생하여 디스플레이 수명을 단축시킬 수 있음.
* **구현 방안:** 다크 모드 기반 위에서 특정 주기(예: 1시간 또는 하루 단위)마다 텍스트 레이아웃의 절대 좌표를 수 픽셀씩 미세하게 이동(Pixel Shifting)시키거나 고정된 UI 그래픽 요소의 위치를 미세하게 변형 및 반전시키는 백그라운드 타이머 로직을 적용함.

## 4. 관제 요원 교대(Handover) 메모 및 체크리스트 영역 추가
* **배경 및 필요성:** 주/야간 근무자가 교대할 때 특정 병실의 주의 환자 상태나 특별 전달 사항 등 인수인계 데이터가 유실되지 않고 상시 공유되어야 연속성 있는 관제가 가능함.
* **구현 방안:** 대시보드 우측 사이드바 하단 구역에 고정형 '인수인계 메모/체크리스트' 컴포넌트를 추가하고, 입력된 데이터는 영속성을 위해 백엔드 DB 또는 브라우저 로컬 스토리지에 동기화되어 새로고침 시에도 유지되도록 설계함.

## 5. 센서/장비 상태 관제 (Heartbeat) 시스템 구축
* **배경 및 필요성:** 특정 병실의 레이더 센서 하드웨어가 고장 나거나 네트워크 단절로 인해 데이터를 전송하지 못하는 상태를 방치하는 것은 환자의 낙상을 감지하지 못하는 것만큼 치명적인 위험을 초래함.
* **구현 방안:** 방(room) 데이터 객체 구조에 `sensorStatus` 필드를 추가하고, 주기적으로 센서와 서버 간 생존 신호(Ping/Heartbeat)를 체크하여 일정 시간 응답이 없는 구역은 카드 UI를 회색으로 블러 처리하거나 빗금 패턴을 적용하여 즉시 장비 장애를 식별할 수 있도록 조치함.

## 6. 비상 연락망 팝업 및 원클릭 호출 인터페이스 추가
* **배경 및 필요성:** 위급 상황 감지 시 단순히 상태 확인 및 상황 해제만 하는 수동적 UI 구조에서 벗어나 관제 요원이 실질적인 초동 조치를 신속하게 취할 수 있어야 함.
* **구현 방안:** 활성 이벤트 알림 카드 내부에 '비상 연락' 버튼을 연동하여 클릭 시 해당 병동 간호사 스테이션의 직통 번호를 팝업으로 표출하거나, WebRTC 기술을 이용해 브라우저에서 다이렉트로 현장 마이크/인터콤과 무선 음성 통화가 연결되는 기능을 구현함.

## 7. 상태별 필터링 및 정렬 기능 추가
* **배경 및 필요성:** 병원의 규모가 확장되어 관제해야 할 방의 개수가 수백 개 단위로 늘어날 경우 정상 상태의 방들을 필터링하고 오직 '이상 징후가 있는 구역'에만 시인성을 집중시킬 필요가 있음.
* **구현 방안:** 메인 화면 상단 그리드 컨트롤 영역에 토글 필터를 배치하여 활성화 시 상태값이 낙상(`fall`), 호흡이상(`breath`)인 카드와 장애가 발생한 오프라인(`offline`) 카드만 상단으로 정렬되거나 화면에 남도록 데이터 배열을 필터링하여 렌더링함.
